
## Used

`ls -tmp`:
- `-t` : sort by modification time, newest first
- `-m` : fill width with a comma separated list of entries
- `-p` : append / indicator to directories

`touch -t 202506012220 test6` : -t Modifying timestamp of a file \
`touch -ht 202506012220 test6` : Use -h to change timestamp on symlink and no the file referenced by it 

`tar -xf testShell00.tar` : -xf to extract archive
